## Admin Account Setup Instructions

1. **Web Interface Method**: The easiest option is to navigate to the `/setup` page where you can easily configure your admin account.

2. **Manual SQL Method**: Alternatively, you can set up the admin account using SQL commands manually. Kindly refer to the database documentation for the necessary SQL commands to create an admin account.

3. **Security Note**: There are no default credentials for security reasons. Make sure to set up your own account during the setup process.

4. **Password Recommendation**: It is highly recommended to use strong passwords to ensure the security of your account.
